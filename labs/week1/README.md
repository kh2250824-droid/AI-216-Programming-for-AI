## What this lab does
Is lab mein maine apna official AI-216 course repository setup kiya hai. Maine seekha ke folder structure kaise banate hain, Git ko initialize kaise karte hain aur files ko GitHub par sahi tarike se kaise push karte hain.

## Key commands used
- `git init` (Repository shuru karne ke liye)
- `git add .` (Changes ko stage karne ke liye)
- `git commit -m "..."` (Changes ko save karne ke liye)
- `git push origin main` (GitHub par upload karne ke liye)
- `git mv` (Files ko rename karne ke liye)
- `git checkout -b` (Naya branch banane ke liye)

## What I learned
- Git aur GitHub ke darmiyan farq aur inka AI projects mein istemal.
- `.gitignore` file ki ehmiyat (un-necessary files ko rokne ke liye).
- Branching workflow aur merge karne ka tarika.

## AI/ML relevance
Version control AI projects ke liye bahut zaroori hai kyunke is se hum apne code aur datasets ke purane versions par wapas ja sakte hain (reproducibility) aur team ke saath mil kar kaam kar sakte hain.
